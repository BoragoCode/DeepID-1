## Prepare Data
1. Download database from [Labeled Faces in the Wild](http://vis-www.cs.umass.edu/lfw/index.html).
2. Extract `lfw.tgz` to `data/`.

```
\-- data
    \-- lfw
        \-- {name}
            |-- {name}_{xxxx}.jpg
    |-- pairsDevTest.txt
    |-- pairsDevTrain.txt
    |-- pairs.txt
```
